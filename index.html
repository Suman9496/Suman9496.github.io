project_lab

Builds from the workbench.

One project shipped and live, one still warming up on Vercel.

Itinerary Builder → PDF
live
React 18 · TypeScript · Vite · React Router · Tailwind CSS · Radix UI · Lucide React · jsPDF + html2canvas · Express · Vitest

A full-stack Vite + React tool for travel consultants to put together a complete trip package — destination, template, dates, travellers, day-by-day activities, hotels, flights, inclusions, visa details, notes, and payment installments — and export the same data straight to a client-ready, multi-page PDF. The live builder form and a hidden print-ready DOM tree share one state object, so the on-screen preview always matches the exported PDF; each page is captured with html2canvas and assembled into the file with jsPDF. An optional Express layer is included for future backend work like auth or CRM integrations.
